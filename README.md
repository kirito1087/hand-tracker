# Neon Hand Control

Live Demo: [https://handtracker1087.netlify.app/](https://handtracker1087.netlify.app/)

This project is a real-time, gesture-based neon drawing and 3D interaction app using hand tracking in the browser.

## Features

- **Draw** neon lines with your index finger.
- **Move** lines by pinching your thumb and index finger.
- **Rotate** lines by making a fist and twisting your wrist.
- **Scale** lines with two open palms.
- **Color selection** and clear canvas.
- **3D Saturn and Galaxy demos** (see navigation buttons).
- **No visible third-party hand tracking branding.**

## How to Use

1. Open the [Live Demo](https://handtracker1087.netlify.app/) in your browser (Chrome recommended).
2. Allow webcam access when prompted.
3. Wait for the loading screen to disappear.
4. Use the color palette to select your drawing color.
5. Use hand gestures to draw, move, rotate, or scale lines as described in the on-screen instructions.
6. Use the navigation buttons to explore the 3D Saturn and Galaxy demos.

## Project Structure

- `index.html` – Main neon drawing app with gesture controls.
- `saturn.html` – 3D Saturn demo with hand tracking.
- `kainat.html` – 3D Galaxy demo with hand tracking.
- All hand tracking is handled with generic, unbranded code.
